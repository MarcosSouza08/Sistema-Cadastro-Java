# Sistema de Cadastro de Usuários em Java

Projeto desenvolvido para praticar conceitos fundamentais de Java e lógica de programação.

## Funcionalidades:

- Cadastro de usuário
- Listagem de usuários
- Busca por CPF
- Atualização de email e senha
- Remoção de usuário
- Validação de email e senha

## Tecnologias utilizadas:

- Java
- ArrayList
- Scanner
- Git e GitHub

## Objetivo:

Consolidar conhecimentos em:
- Estruturas de dados
- Métodos
- Validações
- Organização de código
- Versionamento com Git

## Próximos passos:

- Melhorar validações
- Implementar persistência de dados
- Evoluir para banco de dados

git clone https://github.com/MarcosSouza08/Sistema-Cadastro-Java.git
